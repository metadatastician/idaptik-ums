= RSR philosophy for Universal Modding Studio

The repository applies RSR as evidence discipline: working paths remain
working; generated sources have drift gates; a validator must have negative
tests; formal, runtime, tested, designed and aspirational claims stay
distinct; architectural migration proceeds through reversible, reviewed
seams rather than directory theatre.

Profile separation is an RSR boundary. Universal abstractions are admitted
only when they survive unlike games. Game-specific concepts remain traceable
to their source profile and provenance.
