= idaptik-dlc-vm
:toc:

A 23-instruction reversible virtual machine, packaged as a DLC for the
idaptik game. Written in AffineScript; compiles to WasmGC via typed-wasm.

== What changed from the ReScript original

* *Reversibility is type-checked, not commented.* The ReScript port encoded
  every instruction as `{execute, invert}` — two effectful closures whose
  inverse relationship was a docstring. AffineScript lifts the instruction
  set into a closed sum type; the exhaustiveness checker rejects any
  addition that doesn't update both forward and backward.
* *State access is an effect.* The ReScript `dict<int>` was implicitly
  mutable. AffineScript routes every read/write through the `VmState`
  effect; pure functions cannot touch state.
* *Preconditions are refinements.* `POP target == 0`, `AND ancilla == 0` —
  these were docstrings; they're now refinement predicates (parse-only in
  the current AffineScript checker; live-gated as soon as
  `TRefined`-predicate reduction lands upstream).
* *Modularity per instruction.* `src/instructions/` has 23 files, one per
  instruction. The audit's megafile-split recommendation is structurally
  enforced: it is impossible to write a 2081-LOC LaptopGUI-style file
  here.
* *DLC packaging.* `dlc-manifest.json` declares this as a DLC the game
  loads via its existing DLC loader. The other DLC packs (`dlc-iky`,
  `dlc-reversibley`) collapse from "DLCs that re-implement VM bits" into
  thin imports of this DLC + their own data.

├── src/
│   ├── Instruction.affine     # the closed sum type (23 constructors)

├── tests/

├── dlc-manifest.json

----

== Reversibility classes

Each instruction declares its reversibility class. The proof obligation
varies per class:

[cols="1,3,2"]
|===
| Class | Description | Members

| `:involution`
| `backward = forward`. Self-inverse.
| Swap, Xor, Negate, Flip, Noop

| `:paired`
| `backward(X) = forward(pair(X))`. Two instructions that invert each other.
| Add↔Sub, Push↔Pop, Send↔Recv, Rol↔Ror

| `:additive`
| Forward adds; backward subtracts. State accumulates linearly.
| Load, Store, Mul (into c), Div (into q,r), Recv

| `:precondition-guarded`
| Forward asserts a precondition (e.g. ancilla == 0); backward restores it.
| And, Or

| `:janus-guarded`
| Entry/exit guard registers swap roles under reversal.
| IfZero, IfPos, Loop

| `:compositional`
| Backward runs body in reverse, each instruction inverted.
| Call
|===

For each class, a property test asserts the round-trip identity. See
`tests/properties/reversibility.affine`.
